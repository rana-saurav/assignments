Q5: What is th: DOCTYPE D:claration in HTML"
DOCTYPE (document type declaration) is a specific instruction that declares the version of HTML that a web page is using. It is placed at the very beginning of an HTML document, before the <html> tag, and it helps web browsers interpret and render the page correctly.
