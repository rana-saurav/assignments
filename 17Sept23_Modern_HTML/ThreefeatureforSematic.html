Exploring HTML5 Features

New Semantic Elements

HTML5 introduced several new semantic elements that provide better structure to your HTML, improving accessibility and search engine optimization (SEO). Here are a few examples:

  • <header> - Defines a header for a section or page
  • <footer> - Defines a footer for a section or page
  • <article> - Represents a self-contained piece of content
  • <section> - Defines a section of content
  • <nav> - Represents navigation links
  • <aside> - Represents content tangentially related to the main content

Multimedia Support: Audio & Video

HTML5 makes embedding multimedia content like audio and video much easier with native support for the <audio> and <video> elements.

Here's an example of embedding a video:

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Canvas for Drawing

HTML5 introduced the <canvas> element, which allows you to draw graphics on the fly using JavaScript. This is useful for creating dynamic images, animations, games, and more.

Here's an example of how to use the <canvas> element:

<canvas id="myCanvas" width="500" height="500"></canvas>

HTML5 Forms

HTML5 brought several new input types and attributes to forms, allowing for more dynamic and responsive forms without needing JavaScript.

  • <input type="email"> - Validates email addresses
  • <input type="date"> - Allows users to pick a date
  • <input type="range"> - Creates a slider input
  • <input type="number"> - Restricts input to numbers

Example of a date picker:

<input type="date" name="birthday">
